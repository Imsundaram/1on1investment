# 1 ON 1 INVESTMENT Website

This is the Next.js website for 1 ON 1 INVESTMENT, a real estate and home improvement company.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

-   **Home Page**: Hero section, Featured Properties, Services, Testimonials, Team.
-   **Properties**: Filterable property listings.
-   **Property Details**: Dynamic pages for each property.
-   **Services**: Detailed service descriptions.
-   **Team**: Team member profiles with contact links.
-   **Contact**: Contact form and business info.
-   **Admin Panel**: Basic frontend property management (Mock).
    -   **URL**: `/admin`
    -   **Password**: `admin123`

## Tech Stack

-   **Next.js 15+** (App Router)
-   **Tailwind CSS** (Styling)
-   **Lucide React** (Icons)
-   **TypeScript**

## Customization

-   **Data**: Edit `src/lib/data.ts` to update properties, team members, services, and testimonials.
-   **Colors**: Edit `src/app/globals.css` to change brand colors.
-   **Images**: Currently using Unsplash placeholders. Replace with real images in `public/images/` and update `src/lib/data.ts`.
